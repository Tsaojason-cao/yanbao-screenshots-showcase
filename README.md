# YanBao AI 官网展示页面

雁宝AI私人影像工作室的官方展示网站，采用库洛米主题设计。

## 🌐 在线访问

**GitHub Pages（推荐）**：https://tsaojason-cao.github.io/yanbao-screenshots-showcase/

## ✨ 核心功能展示

- 📍 40个真实机位推荐
- 🎨 12种专业LUT预设
- 🤖 AI影集智能生成
- 🎬 一键成片功能
- 📸 专业相机模式
- 👤 用户自选存储

## 📱 微信兼容性

本网站已针对微信浏览器进行优化：

- ✅ 所有资源使用绝对路径，确保在微信中正常显示
- ✅ 视频支持微信内联播放（X5内核优化）
- ✅ 图片提供WebP和PNG双格式降级方案
- ✅ 完整的微信分享OG标签配置

### 最近修复（2026-01-12）
