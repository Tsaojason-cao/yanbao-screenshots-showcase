## 🎯 技术实现

### 竖版视频生成流程
1. **AI生成竖版镜头**: 使用generate_video工具生成5个9:16竖版镜头
2. **FFmpeg合成**: 统一分辨率到1080x1920，30fps，H.264编码
3. **视频压缩**: 使用CRF 28压缩，减少文件大小
4. **文件清理**: 删除中间文件和旧版本
5. **官网更新**: 更新HTML，优化视频显示样式
6. **GitHub部署**: 推送到GitHub，自动触发Pages部署

### FFmpeg压缩命令
```bash
ffmpeg -i yanbao-promo-vertical.mp4 \
  -c:v libx264 -crf 28 -preset slow \
  -vf "scale=1080:1920" -an \
  yanbao-promo-vertical-compressed.mp4 -y
```

### 压缩参数说明
- **CRF 28**: 质量参数，28为中等压缩（范围0-51，越大越小）
- **preset slow**: 编码速度慢但压缩率高
- **scale**: 确保分辨率为1080x1920
- **-an**: 移除音频轨道（视频本身无音频）
